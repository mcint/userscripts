# scripts/

Installable userscripts. Each is a complete `*.user.js` with a full
`==UserScript==` metadata header and a semver `@version`. Clicking a raw
`.user.js` URL prompts Tampermonkey/Violentmonkey to install.

Start new scripts from [`TEMPLATE.user.js`](TEMPLATE.user.js).

A script earns a place here once it works and is headered. Half-baked ideas
live in [`../drafts/`](../drafts/) until then.
